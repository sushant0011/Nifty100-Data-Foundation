# 📊 Nifty100 Data Foundation

A complete End-to-End Data Engineering project built using Python.

This project extracts financial data from Excel files, validates the data, transforms it, loads it into SQLite, performs SQL analysis, generates analytics reports, and creates visualization dashboards.

---

## 🚀 Features

* Excel Data Loader
* Data Validation
* Data Transformation
* SQLite Database Integration
* Database Validation
* SQL Query Engine
* Business Analytics
* Report Generation
* Dashboard Visualization
